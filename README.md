# Personal Portfolio Website

## Overview

This project is a modern, responsive portfolio website built to showcase my skills, projects, and professional profile. The site is designed to be fast, accessible, and easy to share with recruiters and potential employers.

## Problem

Traditional portfolios are often shared as screenshots or static documents, making it difficult for recruiters to interact with the work. The goal of this project was to create a real, live portfolio website that can be easily accessed and explored online.

## Approach

The website was developed using hand-written HTML, CSS, and JavaScript. Design elements from Figma were translated directly into CSS variables to maintain consistency between the design and implementation.

## Features

* Semantic HTML structure
* Responsive design for desktop, tablet, and mobile devices
* Dark mode support
* Design-system based CSS architecture
* Fast loading performance
* Accessibility-focused implementation
* SEO-friendly structure
* Automatic deployment through GitHub

## Technologies Used

* HTML

* JavaScript
